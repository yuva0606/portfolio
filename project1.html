Ecommerce Backend

This is an Ecommerce BackEnd project where I implemented a microservices structure for core functionalities like product management, inventory, order processing, and user authentication. I used Java and Spring Boot as the backbone, integrating Elasticsearch for fast, reliable search capabilities and Azure Service Bus to enable real-time updates across services. Automation is a big part of my approach, so I developed CI/CD pipelines with GitHub Actions and Azure DevOps to streamline deployment, which significantly reduced release times.

Tech stack

  • Java
  • Spring Boot
  • Docker
  • Kubernetes
  • Github Actions CI/CD
  • Mysql
  • Elasticsearch
  • Rest Api
  • Spring Security, JWT

API Gateway microservice

This is API Gateway for the Ecommerce BackEnd application, which is designed to manage all incoming traffic and route requests to the appropriate microservices. The API Gateway simplifies communication across services by acting as a single entry point, ensuring smooth and secure data flow within the system. It is configured to validate JWT tokens, apply role-based access, and enrich requests with user details, thus enhancing both security and scalability.

User Service

The User Service is a core component of the Ecommerce BackEnd application, responsible for managing user-related operations such as registration, login, and CRUD operations for user profiles. It provides the authentication and authorization backbone for the application by issuing JWT tokens on successful login, which are used to secure access to other services through the API Gateway.

User Service repository link

Product Service

The Product Service is a critical component of the Ecommerce BackEnd application, responsible for handling product-related data and operations. This service manages the addition, retrieval, updating, and deletion of product information and integrates with the Inventory Service to maintain accurate stock levels.

Product Service repository link

Order Service

The Order Service is a critical part of the Ecommerce BackEnd, responsible for managing customer orders. It facilitates order creation and retrieval while ensuring seamless integration with the Product and Inventory Services to maintain accurate stock levels.

Order Service repository link

Inventory Service

The Inventory Service is a crucial part of the Ecommerce BackEnd application, responsible for managing stock levels of products. This service allows for adding, retrieving, and reducing stock quantities, ensuring that stock levels are consistently maintained and available for other services.
