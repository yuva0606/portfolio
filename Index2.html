My Projects

Ecommerce Backend

This is an Ecommerce BackEnd project where I implemented a microservices structure for core functionalities like product management, inventory, order processing, and user authentication. I used Java and Spring Boot as the backbone, integrating Elasticsearch for fast, reliable search capabilities and Azure Service Bus to enable real-time updates across services. Automation is a big part of my approach, so I developed CI/CD pipelines with GitHub Actions and Azure DevOps to streamline deployment, significantly reducing release times.

Tech Stack

  • Java, Spring Boot
  • Docker, Kubernetes
  • GitHub Actions CI/CD
  • MySQL, Elasticsearch
  • REST API
  • Spring Security, JWT

Services

API Gateway

The API Gateway is designed to manage all incoming traffic and route requests to the appropriate microservices. It simplifies communication across services by acting as a single entry point, ensuring smooth and secure data flow within the system. It validates JWT tokens, applies role-based access, and enriches requests with user details, enhancing both security and scalability.

Repository: API Gateway Repository

User Service

The User Service is responsible for managing user-related operations such as registration, login, and CRUD operations for user profiles. It provides authentication and authorization by issuing JWT tokens on successful login, which are used to secure access to other services via the API Gateway.

Repository: User Service Repository

Product Service

The Product Service handles product-related data and operations, including adding, retrieving, updating, and deleting product information. It integrates with the Inventory Service to maintain accurate stock levels.

Repository: Product Service Repository

Order Service

The Order Service manages customer orders by facilitating order creation and retrieval while ensuring integration with Product and Inventory Services to maintain accurate stock levels.

Repository: Order Service Repository

Inventory Service

The Inventory Service manages stock levels of products, allowing for adding, retrieving, and reducing stock quantities to maintain consistency across services.

Repository: Inventory Service Repository

Task Management System

A secure backend application that allows users to register, log in, and manage tasks. Features include JWT-based authentication, role-based access control, and secure CRUD operations.
